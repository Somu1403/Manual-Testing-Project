# Manual Testing Project – E-Commerce Application

## Project Overview

This repository contains a Manual Testing Project for an E-Commerce Web Application.
The objective of this project is to demonstrate **end-to-end manual testing practices, including **test planning, requirement traceability, functional test cases, and validation of critical user flows.

The test scenarios focus on validating the core functionalities of a typical e-commerce platform, such as user registration, login, product search, cart operations, checkout, and order placement.

This project is designed to showcase practical QA skills for entry-level QA / Software Testing roles.

## Test Documentation

The complete testing documentation is available in the Excel file.

The Excel file contains the following sheets:

* **Main Page** – Project dashboard and navigation
* **Test Plan** – Testing strategy, scope, objectives, and deliverables
* **RTM (Requirement Traceability Matrix)** – Mapping between requirements and test cases
* **Functional Test Cases** covering the following modules:

### User Management

* Register User
* Register User with Email
* Login with Correct Credentials
* Login with Incorrect Credentials
* Logout User

### Product & Catalog

* View Product Details
* Product Search
* View Category Products
* View Brand Products
* Add Review on Product

### Cart & Checkout

* Add Products to Cart
* Update Product Quantity in Cart
* Remove Products from Cart
* Add to Cart from Recommended Items

### Order Management

* Place Order (Register During Checkout)
* Place Order (Register Before Checkout)
* Place Order (Login Before Checkout)
* Download Invoice after Purchase

### Additional Validations

* Verify Subscription in Home Page
* Verify Subscription in Cart Page
* Verify Address Details in Checkout
* Verify Scroll Up Functionality
* Search Products and Verify Cart after Login

## Testing Scope

The testing scope includes validation of:

## Testing Methodology

The project follows a **standard Manual Testing lifecycle**:

1. Requirement Understanding
2. Test Planning
3. Test Case Design
4. Requirement Traceability Matrix (RTM)
5. Test Execution Preparation
